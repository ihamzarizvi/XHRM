## Getting started

- Download the latest version of XHRM Starter [here](https://sourceforge.net/projects/XHRM/files/latest/download).

- Prerequisites and environment set up for installing XHRM Starter:
  - [Install on Linux](https://starterhelp.XHRM.com/hc/en-us/articles/6187572000540-Prerequisites-for-installing-XHRM-Starter-in-Linux)
  - [Install on Windows](https://starterhelp.XHRM.com/hc/en-us/articles/6187576427804-Prerequisites-for-installing-XHRM-Starter-in-Windows)

- Install XHRM using the web installer:
  - [XHRM Starter Installation Guide](https://starterhelp.XHRM.com/hc/en-us/articles/5295915003666-XHRM-Starter-Installation-Guide)
  - [XHRM Starter Upgrade Guide](https://starterhelp.XHRM.com/hc/en-us/articles/6937346912402-XHRM-Starter-Upgrade-Guide-For-5x-versions-)

- For further information on how to use the product please refer to the User Guides, Tutorial videos, and FAQs available on [Help Portal](https://starterhelp.XHRM.com)
